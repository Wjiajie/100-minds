# 100-Minds | 思维模型库

掌握跨学科思维模型，做出更好的决策。

## 愿景 (Vision)
建立一个简洁、深度的跨学科认知框架，帮助用户将零散的知识内化为系统的思维模型，提升认知深度与决策质量。

## 实现现状 (Current Status)

### 核心架构
- [x] **Next.js 16 + React 19**：采用最新的 RSC (Server Components) 架构。
- [x] **MDX 驱动**：使用 `next-mdx-remote` 实现内容与设计的完美隔离。
- [x] **禅意设计 (Zen Aesthetic)**：极简的“纸墨”色调，专注内容的深度阅读体验。

### 功能模块
- [x] **思维地图 (Mind Map)**：
  - 基于 MDX 内容的自动化标签提取与分类系统。
  - **星阵 (Constellation) 标签云**：采用阿基米德螺旋排布与 Framer Motion 动效，呈现有机的视觉交互体验。
  - 精简、核心导向的文章摘要描述。
  - 侧重“禅意”与“深度”的学术化文章详情页。
- [x] **术语表系统**：
  - 基于拼音首字母 (A-Z) 的自动排序与索引。
  - 沉浸式居中 Modal 交互。
- [x] **文章渲染**：
  - 支持 `Callout`、`Example`、`Term` 等自定义 MDX 组件。
  - **交互式术语系统**：在 MDX 文档中通过 `<Term>` 标签点击直接触发术语弹窗，实现知识点的深度穿透。
  - 内置 KaTeX 数学公式支持。
  - Noto Serif SC (思源宋体) 衬线字体集成，提升阅读质感。
- [x] **内容校对与标注**：
  - 启动对 `mind-map/` 下 18 篇核心文章的术语标注与校对工程，目前已完成 4 篇重要文章的深度标注。
  - 统一并规范化 `lib/glossary.ts` 中的术语库，移除冗余，确保概念的一致性。
